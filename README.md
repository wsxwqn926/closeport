# w-port 端口管理

## Project setup
```
npm i @w-shao/closeport -g
```

### 关闭端口 prot:端口号
```
w-port close <prot>
```

### 查看端口列表 
```
w-port ls [port]
```
* w-port ls 查看端口列表
* w-port ls [port] 查看具体端口是否被占用及占用程序
